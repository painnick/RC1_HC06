# 블루투스를 활용한 RC 만들기1

모형전자공작 모임 스터디 자료

아두이노와 블루투스를 이용해 RC를 만드는 예제

# 준비물
- Adruino (Nano)
- HC-05/06 블루투스 모듈
- DC 모터(입력 전원에서 동작해야 함. 6V 입력에서는 3V 모터가 타버림)
- L298N 모터 드라이버(다른 모터 드라이버 사용 가능)
- 안드로이드 앱 Arduino Blurtooth Controller(다른 앱 사용 가능)
